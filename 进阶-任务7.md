###题目1： \d，\w,\s,[a-zA-Z0-9],\b,.,*,+,?,x{3},^,$分别是什么?
```
\d 等价于 [0-9]数字字符
\w 等价于 [a-zA-Z_0-9] 单词字符、字母数字下划线
\s 等价于 [\t\n\x0B\f\r] 空白符
[a-zA-Z0-9] 匹配字母数字
\b 单词边界
. 等价于 [^\r\n] 除了回车符和换行符之外的所有字符
* 出现零次或多次（任意次）
+ 出现一次或多次（至少出现一次）
？出现零次或一次（最多出现一次）
x{3}  x出现3次
^ 以XXX开头
$ 以xxx结尾
```
###题目2： 写一个函数trim(str)，去除字符串两边的空白字符
```
function trim(str){
    var reg=/^\s+|\s+$/g;
    return str.replace(reg,"");
}
```
